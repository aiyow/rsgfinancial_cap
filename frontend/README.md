# Condo Portal Frontend

Simple React and Vite frontend connected to the CRUD backend.

```powershell
npm install
npm run dev
```

Set `VITE_API_URL` in `.env` when the backend is not running at `http://localhost:5000`.

- Admin: manage users, units, and assignments.
- Collector: read units and assignments.
- Resident: read only assigned units and personal assignments.
